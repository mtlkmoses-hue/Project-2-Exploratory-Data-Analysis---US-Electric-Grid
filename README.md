# Project-2-Exploratory Data Analysis: 2023 US Electric Grid Disturbances
## Project Description
This project performs a deep-dive Exploratory Data Analysis (EDA) on 2023 electric grid disturbance data. The goal is to understand the "shape and structure" of grid events, investigating how factors like North American Electric Reliability Corporation (NERC) Regions and specific months correlate with Demand Loss (MW) and customer impact.

## Initial Research Questions
1. Which region experienced the highest loss, and does it align with the most frequent events?
2. Are grid disturbances seasonal, and which months are the highest risk?
3. What is the relationship between physical scale (MW) and social impact (Customers Affected)?
4. How many events are recorded as "Unknown," and how does this change the narrative?
