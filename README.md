# Project-2-Exploratory-Data-Analysis-US-Electric-Grid
## Project Description
This project performs a deep-dive Exploratory Data Analysis (EDA) on 2023 electric grid disturbance data. The goal is to understand the "shape and structure" of grid events, investigating how factors like North American Electric Reliability Corporation (NERC) Regions and specific months correlate with Demand Loss (MW) and customer impact.

## Initial Research Questions
1. Which NERC Regions experience the highest frequency of events and Demand Loss?
2. Are there seasonal (monthly) patterns in grid instability?
3. How strong is the correlation between customers affected and actual MW loss?

## Project Requirements
* `pandas` - Data manipulation
* `matplotlib` - Base plotting
* `seaborn` - Advanced statistical visualization
* `numpy` - Numerical operations

## How to Run
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the analysis script: `python grid_eda.py`
4. View the saved `.png` files for the generated insights.
